        • @com.kms.katalon.core.annotation.Keyword static java.lang.String escape(java.lang.String id)

          Convert the given string as "id" parameter into a new string which is suitable as a part of file path.

          This method is declared "non final" intentionally. You can override this "escape" method.

          1. If the id starts with "Test Case/", then the part will be trimmed off. E.g, "Test Cases/TC1" -> "TC1"
          2. If the id starts with "Test Suites/", then the part will be trimmed off. E.g, "Test Suites/TS1" -> "TS1"
          3. A whitespace character is translated into a underbar '_' character. E.g, "Test Cases/main part/TC1" -> "main_part-TC1"
          4. Two or more consecutive whitespace characters are translated into a single underbar '_' character. E.g, "Multiple whitespaces here" -> "Multiple_whitespaces_here"
          5. Several characters, which are NOT allowed in the file path on Windows, are translated into a hyphen '-' character
            • < smaller than
            • > larger than
            • : colon
            • " double quote
            • / forward slash
            • \ back slash
            • | vertical bar
            • ? question
            • * asterisk
          6. Finally, any consecutive hyphen characters are translated into a single hyphen character

          Please note that a forward slash / and a back slash \ will be replaced with a hyphen.

          • E.g, "Test Cases/main/shared/Level 1" -> "main-shared-Level_1"
          Therefore the string value returned will look a flat name, not a tree-like structure anymore.

          If the given id is null, then the pack(id) will silently return null.

          Parameters:
          id
          Returns:
          a string converted from the given id
